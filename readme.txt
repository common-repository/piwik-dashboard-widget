== Description ==

If you are a WordPress user who also uses Piwik to track your website traffic, you're going to be interested in this one! This is a simple plugin that shows your site's analytic summary right on your WordPress Dashboard. Now you don't have to log into an entirely separate system to check you traffic information.

See also http://blog.vedstudio.com/wordpress-plugins/piwik-dashboard-widget/

== Installation ==

1. Install the plugin via WordPress Dashboard
2. At the left side of your Dashboard, click on "Settings", then on "Piwik Widget"
3. Enter the required information in the form - this includes your Piwik server, database username, database password, database name, table prefix, and site id. If you do not have access to this information, try contacting your administrator.
4. Back on your Dashboard, click and drag the widget to position it wherever it's convenient for you! Enjoy!
